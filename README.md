# Express Boilerplate

Yet another express boilerplate.

- Winston logger
  - `warn`+ -> file
  - `info`+ -> console
- Body Parser
  - urlencoded (extended)
  - json (not strict)
- CORS
  - allow all
  - allow all pre-flight
- Helmet
- Nodemon
- Styling Enforcement
  - Eslint: standard JS
  - Prettier: default
    - Single quotes
